Client Project at Bank Central Asia
Year

Sept 2024 - Feb 2025

Role
Data Engineer
Tools & Skills
  • Informatica PowerCenter: ETL development (workflow, mapping, mapplet, job migration)
  • PostgreSQL: Database management, query processing, transformasi, dan validasi data
  • Stored Procedure (SP): Automasi pencatatan activity log
Project Description

Terlibat dalam proyek Master Data Management (MDM) Data Quality di PT Bank Central Asia Tbk dengan menggunakan Informatica Tools. Proyek ini bertujuan untuk meningkatkan kualitas data melalui pengecekan, pembersihan, standarisasi, hingga optimasi proses ETL agar data lebih akurat, konsisten, dan siap digunakan untuk kebutuhan analisis maupun pelaporan.

Contributions
  • Membuat job migrasi data dari berbagai sumber ke database menggunakan Informatica.
  • Mengonversi query SQL menjadi job mapping di Informatica untuk proses validasi Data Quality.
  • Melakukan rekonsiliasi data antara environment Development dan Production guna menjaga konsistensi hasil.
  • Mengembangkan mapplet reusable untuk meningkatkan modularitas dan efisiensi pipeline ETL.
  • Melakukan debugging & perbaikan error job pada fase validasi, serta membuat activity log menggunakan Stored Procedure (SP).
  • Menyiapkan objek deployment ke Production (job, workflow, mapping) dan membangun workflow untuk proses matching data.
  • Melakukan optimasi job mapping di Informatica untuk mempercepat eksekusi dan mengurangi bottleneck sistem.
