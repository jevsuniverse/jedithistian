Project Description

Psikologi adalah tim yang terdiri dari tiga anggota yang berfokus pada pengembangan basis data web khusus untuk psikolog. Platform basis data web ini bertujuan untuk mengumpulkan dan menyimpan data pasien, khususnya pasien psikolog, dalam jumlah besar. Hal ini membantu dalam pengelolaan data pasien secara efisien dan efektif, sehingga menjadi alat yang berharga bagi para profesional di bidang psikologi.
