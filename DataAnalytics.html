Project Description

In this project, I utilized Apache Spark for ramen data analysis, using a series of steps to obtain meaningful visualizations. The process begins with reading the ramen data in CSV format, followed by data conversion and pre-processing to ensure optimal compatibility with Spark. Next, the data is further processed using Spark SQL, enabling comprehensive analysis based on the inherent characteristics of the data set. Leveraging SQL queries, I extracted valuable insights and patterns from the ramen data.
