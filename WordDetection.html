Project Description

This project aims to develop a hate speech detection system using the Python programming language. The main focus of this project is to determine whether a sentence contains hate speech or not. Hate Speech is a form of speech or writing that contains expressions of hatred towards groups or individuals based on attributes such as race, religion, sexual orientation, or others. The methods that will be evaluated in this project include three machine learning algorithms, namely Support Vector Machine (SVM), Naive Bayes, and Decision Trees. Each algorithm has a different approach to understanding and classifying data. Evaluation of each algorithm's performance will be carried out to determine the best algorithm in the context of hate speech detection.
