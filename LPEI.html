Client Project at Lembaga Pembiayaan Ekspor Indonesia / Eximbank
Year

Agust - Sept 2024

Role
Data Engineer
Tools
  • ETL Tools: SQL Server Integration Services (SSIS), SQL Server Management Studio (SSMS)
  • Database: SQL Server
  • Skills: ETL design & troubleshooting, Data transformation, Data validation, Reporting support
Project Description

Mendukung proyek ARRA (Amartha Regulatory Reporting & Analytics) di Eximbank dengan fokus pada pengolahan data melalui proses ETL (Extract, Transform, Load). Proyek ini bertujuan memastikan data yang diproses akurat, konsisten, serta siap digunakan untuk kebutuhan pelaporan regulatori dan internal.

Contributions
  • Membantu penyetelan produk ARRA pada sisi data, termasuk integrasi dan validasi.
  • Membangun serta mengelola job ETL dengan SQL Server Integration Services (SSIS) dan SQL Server Management Studio (SSMS).
  • Melakukan transformasi dan validasi data agar sesuai dengan kebutuhan pelaporan.
  • Melakukan troubleshooting pipeline ETL serta mengoptimalkan performa query.
  • Menyusun dokumentasi teknis untuk mendukung kelancaran proses reporting.
